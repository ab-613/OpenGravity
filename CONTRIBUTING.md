## 🛠️ How to Help
Check the **Issues** tab for labeled "good first issues." Many UI elements are currently placeholders and need real logic:
- Wiring up hardcoded dropdowns.
- Making the top menu bars (File/Edit) functional.
- Adding support for other AI providers (OpenAI, Anthropic).

## 🚀 Process
1. **Fork** the repo and create a branch for your feature.
2. **Submit a Pull Request (PR)** with a clear description of what you fixed or added.
3. Please be patient with reviews as I am currently studying!
